# template-new-react-ts-vit-pnpm-tailwind-less-eslint-stylelint

React + Vite（TS）项目模板，已接入：

- TypeScript
- TailwindCSS
- Less
- ESLint + Stylelint + Prettier
- husky + lint-staged + commitlint
- Vite alias：`@` -> `src`

## Scripts

- dev：`pnpm dev`
- build：`pnpm build`
- preview：`pnpm preview`
- lint：`pnpm lint`
- format：`pnpm format`
- format check：`pnpm format:check`
- commit（交互式）：`pnpm commit`

## Alias

示例：

```ts
import App from '@/App.tsx'
```
